One-time Google setup — takes ~10 minutes
1

Create a Google Cloud Project

Go to console.cloud.google.com → New Project → name it "Vayu Journal" → Create.

2

Enable Google Sheets API

In your project → APIs & Services → Library → search "Google Sheets API" → Enable it.

3

Configure OAuth Consent Screen

APIs & Services → OAuth consent screen → External → Fill in app name "Vayu" and your email → Save. Under Scopes, add:

https://www.googleapis.com/auth/spreadsheets

Under Test users, add your Google email → Save.

4

Create OAuth Client ID

APIs & Services → Credentials → Create Credentials → OAuth client ID → Application type: Web application.

Under Authorised JavaScript origins, add the URL you're running this app from. If opening as a local file, add:

http://localhost

Click Create → copy your Client ID (ends in .apps.googleusercontent.com).

5

Create your Sheets database

Go to sheets.google.com → create a new blank spreadsheet → name it "Vayu Trading Journal".

Copy the Sheet ID from the URL — it's the long string between /d/ and /edit:

docs.google.com/spreadsheets/d/THIS_PART/edit

SNAP Protocol

Use when you feel an override impulse building in-session.
S
Signal Recognition
Name what the body is doing right now. Chest tightening? Hands moving to the keyboard? Urgency building? Just observe it.
N
Name It Out Loud
Say it: "I want to override my plan." "I want to move my stop." "I'm chasing." Naming it activates the witness.
A
Anchor Drop
Three slow breaths. Feel feet on floor. Return to body. The market will not disappear in 30 seconds.
P
Pre-decide the Exit
Your exit was decided before entry. That decision was made by a calm mind. Trust it. If the trade is wrong, the stop handles it.
If SNAP doesn't settle you in 60 seconds — step away from the screen. The trade exists without you watching it.

Day Type Guide

Working criteria to identify the day by ~9:45 AM. These are a starting reference — calibrate the thresholds against your own FRVP read as you log outcomes against them.
📈
Trend Day
Opening range breaks and holds beyond the prior day's value area within the first 30-45 min, with limited overlap back into it. Range expansion clearly above your recent average daily range. POC migrates directionally through the session rather than sitting still. Price accepts new value away from yesterday's — it doesn't keep returning to test it.
〰️
Moderate Day
Some directional movement, but with real two-way rotation — price makes progress then retraces a meaningful chunk before continuing. Range is roughly in line with your recent average, not expanded. Value area overlaps partially with the prior day rather than fully separating from it. A real move exists, but it isn't clean.
⬜
Consolidation Day
Price stays largely inside or tight around the prior day's value area — heavy overlap, POC barely moves. Range is compressed, noticeably below your recent average. Repeated failed breakouts in both directions. This is the day type your own data shows costs you the most when forced — if it's not clearly A+, the honest read is often no trade at all.
⚡
Event Day
A known scheduled event (RBI policy, Fed decision, major macro data, expiry-adjacent geopolitical headline risk) sits in or right before today's session, or VIX has gapped meaningfully overnight. This classification isn't about how price is behaving intraday — it's a pre-market flag based on what's scheduled or already happened before the open. When in doubt about carried/overnight exposure specifically, this is the day type to default to.
If two types feel plausible by 9:45, classify down (toward Consolidation/Event, not up toward Trend) — the cost of under-trading a real trend day is smaller than the cost of over-trading a fake one.
